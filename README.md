## 简介
- 基于 STM32F103C8T6 的 CS2 C4 外观模型，主要用于学习、展示与收藏。
- 开发环境：Keil5 + ST-Link，外部 8 MHz 晶振。
- **[Ver1.0](Ver1.0) 已停止维护**，请移步**最新版本 [Ver4.0](Ver4.0)**。
