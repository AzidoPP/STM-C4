## 简介
- 基于 STM32F103C8T6 的 CS2 C4 外观模型，主要用于学习/展示/收藏。
- 开发环境：Keil5 + ST-Link，外部 8 MHz 晶振。
- V1.0 已停止维护，请跳转到最新版 [Ver4.0](Ver4.0)。
