# 3D 打印说明

## 中文
- `C4_filler` 为填充件，请务必打印。
- 单个填充件尺寸约 190 x 40 x 43 mm（长 x 宽 x 高），以模型文件为准。
- 整体外壳尺寸约 190 x 123 mm（长 x 宽），以模型文件为准。
- 建议填充率 7%~15%，普通材料即可，性价比高。
- 没有打印机可找本地 3D 打印代工，或联系模型提供者 UP 主获取打印件（见下方链接）。
- 不建议选择超低价平台，容易踩坑。
- 总开关外壳文件：`switch_shell`，推荐黑色。

模型来源：
- Bilibili 博主「下界石嘤」提供
- 链接：`https://space.bilibili.com/318570767`(https://space.bilibili.com/318570767)

## English
- `C4_filler` is the filler part and must be printed.
- Filler size is about 190 x 40 x 43 mm (L x W x H). Refer to the model files for exact dimensions.
- Overall shell size is about 190 x 123 mm (L x W). Refer to the model files for exact dimensions.
- Recommended infill: 7%–15%. Standard materials are fine and cost-effective.
- If you don't have a printer, use a local 3D printing service or contact the model provider (see the link below) for printed parts.
- Avoid ultra-cheap platforms; the quality and pricing are often unreliable.
- Main switch shell file: `switch_shell`, black color recommended.
