# EasyEDA 工程说明 / EasyEDA Project Notes

## 中文

本目录收录 EasyEDA 原始工程文件，当前文件名为 `ProPrj_CS2-C4_V4.1.epro2`。

尽管文件版本标注为 **V4.1**，其设计内容与本仓库 `Ver4.0` 目录对应的硬件版本兼容，可用于 `Ver4.0` 的工程查阅、复核与后续编辑。

如需在线查看或对照硬件开源页面，亦可访问 OSHWHUB：
https://oshwhub.com/azidopp/CS2-C4

## English

This directory contains the original EasyEDA project file: `ProPrj_CS2-C4_V4.1.epro2`.

Although the file is labeled **V4.1**, its design remains compatible with the hardware version documented under `Ver4.0` in this repository, and can be used for review, verification, and subsequent edits for `Ver4.0`.

For online access and cross-reference, you may also visit the OSHWHUB page:
